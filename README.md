# 🔋 AI-Based Transformer Health Monitoring System

## 📌 Overview

The **AI-Based Transformer Health Monitoring System** is a Machine Learning-powered predictive maintenance application designed to monitor transformer operating conditions and predict health status.

The system analyzes important electrical parameters such as temperature, voltage, current, and oil level to classify transformer conditions into:

- ✅ Healthy
- ⚠️ Warning
- 🔴 Critical

The project combines **Electrical Engineering concepts, Machine Learning, and Streamlit dashboard development** to create an intelligent monitoring solution.

---

## 🚀 Features

✅ Transformer condition prediction using Machine Learning  
✅ Real-time parameter input through an interactive dashboard  
✅ Health status classification  
✅ Simple and user-friendly Streamlit interface  
✅ Predictive maintenance approach for power systems  
✅ Helps identify possible transformer failures early

## 📊 Input Parameters

| Parameter | Description |
|-----------|-------------|
| Temperature | Transformer operating temperature (°C) |
| Voltage | Transformer voltage level (V) |
| Current | Load current (A) |
| Oil Level | Transformer oil percentage (%) |
